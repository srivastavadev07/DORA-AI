# 🤖 DORA AI

An AI-powered desktop assistant built with Python that can understand voice commands, answer questions using AI, open applications, automate Windows tasks, capture screenshots, and more.

---

## ✨ Features

- 🎤 Voice Recognition
- 🤖 AI Chat using OpenRouter
- 🌐 Open Websites
- 📂 Launch Applications
- 🔒 Lock Windows
- 📸 Screenshot Capture

---

## 🛠 Tech Stack

- Python
- OpenRouter API
- SpeechRecognition
- Pillow
- PyAudio
- pygame
- dotenv

---

```bash
git clone https://github.com/srivastavadev07/DORA-AI.git
